**DirectoryPress** professional directory plugin for Wordpress.

DirectoryPress is a fully featured directory theme for Wordpress, it allows you to turn your standard wordpress blog into a powerful online link directory.

Some features:

* Utilises WordPress pages, or posts, to create a link directory layout
* Custom Advertising Spaces (Google Adsence and normall banners)
* Domz extractor allowing you to import listings from domz website.
* Various methods available for listing products
* Full Google Adsence Integration for directory link clicks and selections

== Installation ==

1. Unzip into your `/wp-content/themes/` directory. If you're uploading it make sure to upload
the top-level folder. Don't just upload all the php files and put them in `/wp-content/themes/`.
1. Activate the plugin through the 'Appearance' menu in WordPress

For upgrading please use check our website http://www.directorypress.net/
If you have to upgrade manually simply repeat the installation steps.
